# 《啾啾和大力》巨大物品系列选题库 V1.0

> 用途：用于《啾啾和大力》后续短视频选题、分镜、关键帧、视频提示词和评论区互动规划。  
> 当前阶段：优先执行 **巨大日常物品系列**。  
> 生产方式：每条建议按 **3 个 5 秒镜头** 生成，再剪辑成 10-15 秒短视频。  
> 核心原则：**一条视频只讲一个错位笑点；镜头替代对白；结尾停在反应。**

## 1.1 巨大快递盒

### 推荐标题

> 盒子刚打开，啾啾已经住进去了

### 备选标题

- 大力还在搬，啾啾已经入住了
- 这不是快递盒，是啾啾的新家
- 盒子还没拆，啾啾先搬家了

### 核心笑点

大力想认真完成任务，把巨大快递盒挪开；啾啾却把盒子当成新家，直接钻进去住下。

### 结尾类型

闯祸型 / 温暖型都可以。

### 三镜头结构

| 镜头 | 内容 | 结尾反应 |
|---|---|---|
| 镜头1 | 门口玄关出现一个像小房子一样的巨大快递盒，啾啾和大力抬头愣住 | 大力抬起小爪子，像要指挥 |
| 镜头2 | 大力认真指挥啾啾推盒子，啾啾却被盒子里面吸引 | 啾啾把头探进盒子 |
| 镜头3 | 啾啾直接钻进盒子里趴下，大力站在外面愣住 | 啾啾满足，大力帽子歪 |

### 画面重点

- 快递盒要干净、无品牌、无文字。
- 盒子像小房子，但不能过于复杂。
- 啾啾钻盒子的动作要可爱，不要卡住。
- 大力站在盒子外面显得很小，反差更强。

### 风险控制

```text
不要出现品牌logo，不要出现快递单文字，不要乱码文字，不要纸箱脏乱，不要啾啾被卡住，不要完整人类。
```

### 评论区互动

```text
啾啾这是在帮忙，还是在搬家？
A：帮忙，只是方式特殊
B：搬家，已经入住
C：大力白指挥了
D：盒子归啾啾了
```

### 推荐指数

> 10/10。最适合下一集做。

## 1.2 巨大雨伞

### 推荐标题

> 伞一打开，啾啾以为房子到了

### 备选标题

- 下雨了，它们直接住进伞里
- 这把伞不是用来撑的，是用来住的
- 大力还在研究，啾啾已经躲好了

### 核心笑点

一把普通雨伞在小动物视角里像巨大帐篷。雨伞轻轻打开后，两只从紧张变成安心躲雨。

### 结尾类型

温暖型。

### 三镜头结构

| 镜头 | 内容 | 结尾反应 |
|---|---|---|
| 镜头1 | 地上一把合上的巨大雨伞，啾啾和大力靠近观察 | 两只抬头看伞柄 |
| 镜头2 | 雨伞轻轻弹开，像帐篷一样撑起 | 啾啾和大力被罩在伞下愣住 |
| 镜头3 | 外面有轻微雨声，两只安静坐在伞下 | 啾啾和大力靠在一起，温暖收尾 |

### 画面重点

- 雨伞要干净、低饱和、温暖。
- 打开动作要轻，不要弹飞角色。
- 伞下空间像小帐篷，有治愈感。
- 结尾可以偏温暖，不一定搞笑翻车。

### 风险控制

```text
不要雨伞猛烈弹开，不要把角色弹飞，不要危险感，不要暴雨灾难感，不要高空边缘，不要角色受伤。
```

### 评论区互动

```text
这把伞对它们来说像什么？
A：帐篷
B：小房子
C：移动城堡
D：啾啾的新窝
```

### 推荐指数

> 9/10。适合放在快递盒后面，用来拉开内容调性。

## 1.3 巨大拖鞋

> 大力说这是船，啾啾已经上去了

| 镜头 | 内容 | 结尾反应 |
|---|---|---|
| 镜头1 | 一只巨大干净拖鞋横在地毯上，像洞穴一样 | 啾啾和大力同时看向鞋口 |

| 镜头3 | 啾啾钻进拖鞋，只露出圆脸和粉色背心 | 大力在外面愣住，帽子歪 |

### 画面重点

- 拖鞋必须干净可爱，不要脏乱。
- 鞋口像一个小洞穴，啾啾探头要可爱。
- 大力站在鞋口外，显得特别小。

### 风险控制

### 推荐指数

## 1.4 巨大毛线球

### 推荐标题

> 毛线球一滚，大力先被带走了

### 备选标题

- 啾啾只是拨了一下，大力先慌了
- 大力说它能控制，下一秒帽子歪了
- 这不是毛线球，是大力的过山车

### 核心笑点

啾啾轻轻拨毛线球，大力自信指挥，结果毛线轻轻挂住大力，把它带得踉跄几步。

### 结尾类型

闯祸型。

### 三镜头结构

| 镜头 | 内容 | 结尾反应 |
|---|---|---|
| 镜头1 | 巨大毛线球滚到地毯上，啾啾眼睛亮了 | 大力抬爪表示“交给我” |
| 镜头2 | 啾啾伸爪轻轻拨毛线球，大力在旁边指挥 | 毛线球轻轻晃动 |
| 镜头3 | 毛线轻轻挂住大力，大力被带歪几步 | 大力帽子歪，啾啾发愣 |

### 画面重点

- 毛线球颜色柔和，适合治愈风。
- 毛线只轻微挂住大力，不要缠住脖子或身体太紧。
- 大力是被轻轻带歪，不是被拖走受伤。

### 风险控制

```text
不要危险缠绕，不要勒住角色，不要角色被拖很远，不要毛线球高速滚动，不要帽子飞起来，不要两顶帽子。
```

### 评论区互动

```text
大力这次出了多少力？
A：100分，精神上很努力
B：80分，至少站前面了
C：50分，被毛线带走了
D：0分，帽子又歪了
```

### 推荐指数

> 8.5/10。动作喜剧感强，但要控制安全感。

## 3. 第三梯队：有潜力，但暂不优先

---

## 3.1 巨大奶茶

### 推荐标题

> 这不是奶茶，是大力的游泳池

### 核心笑点

奶茶杯在小动物视角里像甜味深井 / 小池塘。大力想闻一口，差点在杯边打滑。

### 风险

- 液体容易被生成成危险场面。
- 角色可能掉进去，产生溺水感。
- 杯身容易出现乱码文字和品牌。

### 建议

后期角色和视频稳定后再做。若要做，杯身必须无字无品牌，动作控制在杯边观察，不掉入液体。

---

## 3.2 巨大披萨

### 推荐标题

> 披萨盒一打开，大力迷路了

### 核心笑点

披萨对大力来说像一片圆形广场。大力站在披萨边缘找不到出口，啾啾在旁边看得很认真。

### 风险

- 食物细节容易脏乱油腻。
- 披萨盒可能出现品牌和乱码。
- 画面可能变成食物广告感。

### 建议

后期再做。保持披萨干净、低油光、无品牌文字。

---

## 3.3 巨大手机

### 推荐标题

> 大力不是看导航，它走在导航上

### 核心笑点

手机屏幕对大力来说像巨型地图。大力站在屏幕上，以为自己在走导航。

### 风险

- 屏幕容易出现乱码 UI。
- 手机品牌、图标、文字容易失控。
- 反光和屏幕内容可能干扰角色。

### 建议

暂不优先。若做，屏幕只保留简单发光地图色块，不出现具体文字和品牌。

## 5. 选题判断标准

每次选择下一集前，先检查：

| 检查项 | 是否通过 |
|---|---|
| 物品是否一眼看懂？ | 是 / 否 |
| 物品是否适合巨大化？ | 是 / 否 |
| 是否能在 3 个 5 秒镜头里讲完？ | 是 / 否 |
| 是否只讲一个错位笑点？ | 是 / 否 |
| 是否能体现啾啾或大力的性格？ | 是 / 否 |
| 是否不依赖对白也能看懂？ | 是 / 否 |
| 是否没有强危险感？ | 是 / 否 |
| 是否不容易出现品牌/乱码？ | 是 / 否 |
| 是否不会让角色变形或道具复制？ | 是 / 否 |

如果有两项以上为“否”，该选题暂缓。
