## 1. 统一角色卡

### 啾啾

```text
白色银渐层小母猫，卡通化大头短肢四足体态，圆脸，大而清晰的深色眼睛，
柔软块状绒毛，始终穿粉色绒绒背心和白色包边；
好奇、先行动、容易温和偏离计划；
禁止换品种、换衣服、增加项圈、帽子或复杂配饰，禁止真实成年猫长腿比例。
```

### 大力

```text
奶茶色小男仓鼠，圆头、短手短脚、身体小而有弹性，
站立高度为啾啾肩高 35%—45%，含帽最高不超过 50%；
始终只有一顶低饱和牛仔蓝小棒球帽，不穿其他衣服；
认真、爱指挥、努力解决问题；
禁止变大、老鼠化、长尾巴、穿衣服、第二顶帽子或帽子成为剧情道具。
```

## 9. 母版验收

- [ ] 与当前人工通过角色图身份一致；
- [ ] 大力为啾啾肩高 35%—45%；
- [ ] 大力只有一顶蓝帽且帽子不承担剧情；
- [ ] 巨物 > 啾啾 > 大力；
- [ ] 同一透视平面；
- [ ] 一张母版只有一个用途；
- [ ] 动作和表情在手机尺寸可读；
- [ ] 无额外肢体、畸形、品牌、文字、水印和完整人类；
- [ ] 无真实伤害和危险感；
- [ ] 已通过禁重复检查。

任一项失败，移入候选目录，不得作为强参考。
