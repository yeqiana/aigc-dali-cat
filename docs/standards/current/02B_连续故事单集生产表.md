## 6. 角色和场景连续性

### 角色 P0

- 啾啾：白色银渐层小母猫，粉色绒绒背心、白色包边；
- 大力：奶茶色小男仓鼠，约啾啾肩高 35%—45%，唯一牛仔蓝帽，不穿衣服、不长尾巴；
- 两者是搭档；
- 大力有明确计划或判断；
- 啾啾的行动是对目标的偏离、误解或过度执行。

## 10. B 模式 P0

- [ ] 已通过 A-Lite 升级门槛；
- [ ] 已通过禁重复检查；
- [ ] 没有重复 A-Lite 原测试剧情；
- [ ] 当前单集只承担一个故事节点；
- [ ] 第一帧已经处于新异常状态；
- [ ] 2.5 秒内看懂本集目标；
- [ ] 无声观看能理解主要因果；
- [ ] 角色身份、比例、服装和唯一帽子连续；
- [ ] 与上集尾帧空间和物体状态衔接；
- [ ] 1—2 段动态均有预备、主动作和结果；
- [ ] 有一个有效结果、翻转或循环；
- [ ] 第五集提供情绪或关系回报；
- [ ] 无真实伤害、品牌、随机文字、完整人类和额外角色；
- [ ] 声音与动作同步并保留短静默；
- [ ] 成片无水印并达到正式导出参数。

最终结论：

```text
正式发布通过 / 修改后发布 / 不通过
```
