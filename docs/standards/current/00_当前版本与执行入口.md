## 1. 唯一执行起点

任何新单集必须从本文件开始，不得直接复制旧单集、不从历史分支复制模板，也不得只读取某一份规范后自行推断。

固定顺序：

```text
先查已发布内容与禁重复清单
→ 选择 A-Lite / B / C 模式
→ 复制对应单集生产表
→ 读取角色、比例、母版和安全 P0
→ 制作关键帧与动态段
→ 使用对应模式验收
→ 发布并写回复盘数据
```

## 2. 规范优先级

### 2.1 全模式共同 P0

以下文件对 A-Lite、B、C 全部生效：

1. `01_最终执行规范.md`：角色、世界观、画面、安全和全局生产边界；
2. `03_母版清单.md`：当前可用强参考与候选母版状态；
3. `04_全量母版生成提示词.md`：母版和单集图像的视觉底座；
4. `05_尺寸与比例规范.md`：角色和巨物比例硬约束；
5. `10_通用场景母版验收标准_V1.1.md`：通用场景控制包和固定场景母版的 P0 验收；
6. `11_通用室内场景平面动线图生成规范.md`：S00 平面动线、机位和轴线控制；
7. `docs/strategy/已发布内容与禁重复清单.md`：立项前去重硬约束。

角色身份、服装、比例、安全、无品牌、无随机文字、无完整人类、无额外角色等 P0，不因制作模式降低。

### 2.2 A-Lite 模式

适用：路线图第 1—30 集、尚未验证的新题材、主要使用免费额度的测试内容。

执行文件：

1. `08_A-Lite低成本生产规范V1.0.md`；
2. `02A_A-Lite单集生产表.md`；
3. `09_成片评审与发布验收规范.md` 的 A-Lite 验收部分。

冲突处理：

```text
A-Lite 在时长、动态段数、关键帧数、翻转数量、声音复杂度和尝试上限方面，
优先于 01、07、09 中的 B/C 常规要求。
```

A-Lite 固定边界：

- 6—8 秒；
- 1 段 3—5 秒动态；
- 1—2 张关键帧；
- 1 个核心动作；
- 1 个轻反转或循环接点即可；
- 单动态最多尝试 4 次；
- 使用简化声音时间轴；
- 不要求 4—6 个生成片段；
- 不要求两次视觉翻转；
- 不要求完整六阶段管弦乐正文。

### 2.3 B 模式

适用：第 31—70 集，且相关动作结构已经通过 A-Lite 数据验证。

执行文件：

1. `01_最终执行规范.md`；
2. `02B_连续故事单集生产表.md`；
3. `07_配乐与音效执行规范.md` 的 B 模式部分；
4. `09_成片评审与发布验收规范.md` 的 B 模式部分。

B 模式固定边界：

- 8—12 秒；
- 1—2 段动态；
- 2—3 张关键帧；
- 每集只完成五集故事中的一个剧情节点；
- 允许 1—2 个翻转，但不为凑结构强行增加；
- 同一五集故事复用场景和道具；
- 必须说明与对应 A-Lite 测试内容的差异；
- 未通过升级门槛，不得直接进入 B。

### 2.4 C 模式

适用：第 71—100 集或数据胜出题材的精品重制。

执行文件：

1. `01_最终执行规范.md`；
2. `02C_精品重制单集生产表.md`；
3. `07_配乐与音效执行规范.md` 的 C 模式部分；
4. `09_成片评审与发布验收规范.md` 的 C 模式部分。

C 模式固定边界：

- 10—15 秒，明确高潮时可到 15—20 秒；
- 2—4 段动态；
- 3—6 张关键帧；
- 完整动作链和明确情绪回报；
- 付费工具使用前必须通过预算与数据门槛；
- 不得仅因为“想做精品”跳过 A-Lite/B 验证。

## 4. 新单集立项门禁

新单集必须同时满足：

- [ ] 已在禁重复清单中检索；
- [ ] 已填写“道具 + 核心运动 + 大力目标 + 啾啾偏离 + 结果”题材指纹；
- [ ] 未命中永久禁区；
- [ ] 与最近 10 条内容不存在三项及以上相同；
- [ ] 已选择 A-Lite / B / C；
- [ ] 已复制正确模板；
- [ ] 已确认使用的母版均为 P0 通过强参考，或明确标记为身份/构图弱参考；
- [ ] 已写明尝试上限和降级方案；
- [ ] 已确定无声观看可以理解的视觉因果。

任一项未完成，不进入图片或视频生成。

## 6. 发布前最低要求

### A-Lite

- 第一帧已经发生异常；
- 1 秒内看见巨物与运动方向；
- 2 秒内看懂大力目标或啾啾行动；
- 全片只有一个核心动作；
- 有一个清楚结果、轻反转或循环接点；
- 通过角色、比例、安全和无文字 P0；
- 具备最小声音因果和短静默；
- 无水印，导出达到正式发布参数。

### B

- 单集节点独立可懂；
- 能看出本集属于五集故事的哪个节点；
- 与 A-Lite 原测试不重复；
- 目标、偏离、结果成立；
- 有 1—2 个有效翻转或一个强循环；
- 通过 B 模式声音和成片验收。

### C

- 数据与预算门槛已通过；
- 精品化带来明确视觉或情绪提升；
- 完整动作链成立；
- 高潮和情绪回报清楚；
- 通过完整声音、连续性和技术交付验收。

## 7. 一票否决

- 重做“帽子搭便车、巨大西瓜追逐、快递盒为家”或其高相似版本；
- 蓝帽成为车、船、滑板、失踪物、被追逐物或整集任务道具；
- 使用错误模板；
- A-Lite 被强制扩写成 4—6 段或两次翻转；
- B/C 未通过数据门槛直接制作；
- 大力变大、长尾巴、穿衣服、老鼠化或出现第二顶帽子；
- 啾啾换品种、换背心或变成成年真实猫比例；
- 真实伤害、压伤、夹伤、危险碰撞或恐怖感；
- 依赖字幕才能理解；
- 正式文件带水印、品牌、随机文字或完整人类；
- 未复盘前批量生产下一批。
