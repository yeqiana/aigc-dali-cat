## 1. 项目定位

《啾啾和大力》是无对白、夸张、荒诞、快速节奏的动画喜剧 IP。正式导演语言统一称为“经典无对白剧场动画式喜剧语法”，不得复制具体作品的画面、角色、分镜、旋律、配乐或音效。

固定喜剧公式：

```text
异常已经发生 → 大力认真分析、准备或执行 → 啾啾不按计划直接行动
→ 世界发生荒诞的物理变化 → 大力延迟反应 → 结尾再补一个尾梗
```

## 2. 唯一正式生产流程

```text
立项与禁重复
→ Storyboard（故事板）导演表
→ Storyboard 图
→ Storyboard P0
→ 正式 Keyframe（关键帧）计划
→ Keyframe 生成
→ Keyframe P0
→ 分段视频
→ 配乐音效与剪辑
→ 成片 P0
→ 发布与数据复盘
```

Storyboard 必须先于 Keyframe；未通过 Storyboard P0，不得进入正式 Keyframe 生成。
