## 8. 三阶段 P0

### Storyboard P0

- [ ] 钩子、揭晓、升级、第一次反转、延迟反应、尾梗职责清楚；
- [ ] 3 秒内出现第一次反转；
- [ ] 动作方向、角色位置、道具状态和声音先后连续；
- [ ] 不依赖字幕；角色身份、比例、安全和无文字通过。

### Keyframe P0

- [ ] 当前强参考使用正确；
- [ ] 角色身份、服装、比例、地面透视和道具状态连续；
- [ ] 高风险因果与首尾状态已独立锁定；
- [ ] 无额外肢体、随机文字、水印、品牌或真实伤害。

### 成片 P0

- [ ] 第一秒兑现 SB01/SB02；3 秒内第一次反转；
- [ ] 延迟反应和尾梗独立可读；
- [ ] 无无效铺垫、因果丢失、声音先后错误；
- [ ] 保留短静默，结尾无完整终止和弦；
- [ ] 未为生成方便删除关键喜剧节点。
