## 11. 四次尝试止损

| 尝试 | 只允许处理的事项 | 结果 |
|---:|---|---|
| 1 | 修正最明显的动作方向、比例或道具结构之一 | |
| 2 | 减少一个同时运动的角色 | |
| 3 | 降低速度、缩小位移、固定镜头 | |
| 4 | 最后一次；失败后停止消耗 | |

第 4 次失败后的降级方案：

- [ ] 双角色运动改为单角色运动；
- [ ] 角色运动改为道具运动；
- [ ] 高速动作改为缓慢动作；
- [ ] 使用两张静态帧表现开始和结果；
- [ ] 用剪辑推镜、速度变化或轻震动补足；
- [ ] 用声音表现不稳定的中间过程；
- [ ] 更换下一候选选题。

## 12. A-Lite P0 验收

- [ ] 已通过禁重复检查；
- [ ] 第一帧已经发生异常；
- [ ] 1 秒内看见巨物和运动方向；

- [ ] 无声观看能理解开始和结果；
- [ ] 啾啾身份、脸型、毛色和粉色背心连续；
- [ ] 大力身份、比例和唯一蓝帽连续；
- [ ] 大力没有衣服、长尾巴、老鼠化或体型放大；
- [ ] 巨物 > 啾啾 > 大力；
- [ ] 两个角色处于同一透视地面；
- [ ] 巨物与背景有色彩或明暗分离；
- [ ] 角色不是写实宠物摄影；
- [ ] 没有真实伤害和危险感；
- [ ] 没有文字、品牌、完整人类和额外角色；
