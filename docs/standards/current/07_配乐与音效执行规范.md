# 《啾啾和大力》配乐与音效执行规范 V1.0｜锁定版

> 状态：`main` 当前声音生产硬约束  
> 生效日期：2026-07-11  
> 适用范围：所有单集的最终视频提示词、视频生成、剪辑、混音和发布验收

## 1. 规范目标

本系列统一使用复古无对白动画喜剧的动作同步管弦乐。配乐不是持续铺底的装饰，而是与镜头、角色表演和物体运动共同承担叙事。

每条最终视频提示词必须包含完整的“配乐与音效设计”正文，逐时间段写明：

- 管弦乐配器与节奏；
- 角色和巨物的动作同步点；
- 关键因果音效；
- 高潮前完整静默；
- 第一次翻转重音；
- 第二次翻转递进；
- 结尾反应音；
- 未解决循环音型；
- 声音层级和负面约束。

禁止只写“轻快音乐”“加入动作音效”“背景音乐逐渐加速”或一行“声音参考”。缺少完整声音正文的提示词不得进入视频生成。

## 2. 全局声音风格

```text
采用复古无对白动画喜剧的动作同步管弦乐配乐。配乐紧密跟随角色动作和物体运动，通过弦乐、木管、少量铜管及打击乐代替对白表达情绪。
```

禁止使用：

- 流行歌曲、歌词和人声演唱；
- 旁白、对白和解释剧情的动物叫声；
- 电子舞曲、史诗预告片音乐和恐怖弦乐；
- 持续铺满全片的背景音乐；
- 持续轰鸣低音和沉重伤害音效；
- 从头到尾保持同一速度；
- 破坏循环的完整终止和弦。

## 3. 标准节奏曲线

```text
快速异常事件
→ 轻快蓄力
→ 动作加速
→ 高潮前突然静默
→ 管弦乐反转重音
→ 第二次翻转快速收尾
→ 未完全解决的循环音型
```

### 3.1 开头事件

- 音乐必须立即进入事件，不播放平静前奏。
- 巨物运动优先使用低音弦乐拨奏、低音管或短促圆润重音表现体积与方向。
- 角色脚步、打滑和受惊反应使用短木管、木块、长笛上滑等轻巧音色同步。

### 3.2 目标与蓄力

- 弦乐拨奏保持轻快、规律并逐渐加速。
- 大力的认真、用力和快速小步优先用低音管短促重复音型表现。
- 啾啾锁定目标时使用单簧管或长笛逐步升高的短音建立预期。
- 此阶段不得提前出现高潮重音。

### 3.3 主动作

- 起跳、扑出、弹起或快速前冲对应短促上行音阶。
- 动作落点使用轻拨弦、木块或柔和打击乐，不使用沉重撞击声。
- 主动作达到第一次速度峰值，但保持安全、可爱，不营造真实危险感。

### 3.4 高潮前静默

- 翻转前以快速升高并突然截断的弦乐紧张音结束蓄力。
- 管弦乐必须完整停止0.15–0.25秒。
- 静默前只允许短暂保留与因果直接相关的细小声音，如绳线绷紧、物体轻颤。
- 静默期间不得加入持续BGM、对白、叫声或无关环境声。

### 3.5 第一次翻转

- 翻转第一帧使用清楚、短促的管弦乐错位重音。
- 物体改变方向时，音乐方向、音高走势或节奏必须同步改变。
- 可使用弦乐快速下滑、低音管下行和轻钹点表达预期被打破。
- 角色发现翻车时突然放慢配乐，用极短低音管滑音代替对白反应，通常不超过0.4秒。

### 3.6 第二次翻转与循环

- 笑点接近时使用三次递进短音，节奏一次比一次快。
- 笑点落点使用轻巧清脆的拨弦、木琴或材质摩擦音，不使用疼痛音效。
- 结尾反应以极短木管尾音表达，不拖长停顿。
- 最后0.3秒不得使用完整终止和弦；保留未解决音型，使节拍、音高重心和速度能够接回开头。

## 4. 动作同步原则

配乐必须跟随动作，但不能为每个细小动作堆叠重音。每集只把主要重音放在以下四类节点：

1. 巨物或异常事件突然进入；
2. 主动作明确落点；
3. 第一次翻转发生；
4. 第二次翻转或结尾笑点兑现。

单集必须逐项填写：

```text
巨物速度或方向变化 → 对应的节奏、音高和配器变化
角色预备动作 → 对应的蓄力音型
角色爆发动作 → 对应的上行、加速或重音
动作落点 → 对应的轻打击或拨弦落点
高潮前截断 → 对应的完整静默
第一次翻转 → 对应的错位重音和音乐方向变化
第二次翻转 → 对应的三次递进短音
结尾反应 → 对应的极短木管尾音
```

## 5. 声音层级与混音

声音优先级固定为：

```text
关键因果音效 > 高潮前完整静默 > 动作同步管弦乐 > 极弱环境底噪
```

- 管弦乐不得遮住巨物运动、绳线绷紧、回弹、开合、摩擦等关键因果音效。
- 背景音乐不得填满所有节奏点。
- 全片无对白、无旁白、无歌词、无“meow”等无意义字幕或叫声解释剧情。
- 最终混音真峰值不高于-1 dBFS。
- 音频导出使用AAC、48kHz、立体声、192kbps。

## 7. P0验收

- [ ] 最终视频提示词包含完整配乐与音效正文，不是一行声音备注。
- [ ] 配乐为动作同步管弦乐，并随巨物和角色动作改变节奏、音高或配器。
- [ ] 开头音乐立即进入异常事件，没有平静前奏。
- [ ] 主动作前后具有清楚的加速和落点。
- [ ] 第一次翻转前具有0.15–0.25秒可感知的完整静默。
- [ ] 第一次翻转具有错位重音和音乐方向或节奏变化。
- [ ] 第二次翻转使用递进短音快速收尾。
- [ ] 结尾没有完整终止和弦，音型能够接回开头。
- [ ] 全片无对白、旁白、歌词和无意义叫声。
- [ ] 关键因果音效没有被管弦乐遮挡。
- [ ] 最终真峰值不高于-1 dBFS，音频达到AAC、48kHz、立体声、192kbps。

任意一项未通过，不进入正式视频生成或发布。

## 8. 已验收标准示例

“帽子搭便车”的完整6秒配乐与音效设计是本规范的首个合格样例。其对应关系为：

| 时间 | 画面节点 | 配乐与音效标准 |
|---:|---|---|
| 0.0–0.8秒 | 毛线球进入 | 低音弦乐快速拨奏、低音管重音、木管与木块同步打滑、长笛受惊上滑 |
| 0.8–2.0秒 | 大力拉线、啾啾锁定 | 弦乐拨奏逐渐加速、低音管重复音型、单簧管或长笛三个升高短音 |
| 2.0–3.0秒 | 啾啾扑线 | 长笛与小提琴上行音阶、轻拨弦和木块动作落点 |
| 3.0–3.4秒 | 毛线绷紧 | 弦乐升高后突然截断、0.15–0.25秒完整静默 |
| 3.4–4.7秒 | 毛线球反向滚回 | 弦乐下滑、低音管下行、轻钹错位重音、短促疑问滑音 |
| 4.7–6.0秒 | 帽子被勾走 | 小提琴拨奏和木琴三次递进、清脆拨弦与布料摩擦、长笛上行、未解决循环尾音 |

后续单集可以更换巨物、动作、具体乐器组合和时间点，但不得降低本规范第1–7节的硬要求。
