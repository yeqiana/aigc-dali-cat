> 生效日期：2026-07-11  
> 适用：B 连续故事和 C 精品重制  
> A-Lite 不执行本文件的完整要求，改用 `08_A-Lite低成本生产规范V1.0.md` 的简化声音时间轴。

---

## 1. 共同声音边界

- 无对白、旁白、歌词和解释剧情的动物叫声；
- 不使用流行歌曲、电子舞曲、史诗预告片音乐和恐怖弦乐；
- 关键因果音效优先于背景音乐；
- 必须保留 0.15—0.25 秒可感知静默；
- 不使用真实疼痛、重击和受伤声音；
- 结尾不使用沉重完整终止；
- 正式混音真峰值不高于 -1 dBFS；
- AAC、48 kHz、立体声、192 kbps。

声音层级：

```text
关键因果音效 > 静默 > 动作同步配乐 > 极弱环境底噪
```

## 2. B 模式标准声音

B 每集只承担一个故事节点，声音不需要强行六阶段全部出现，但必须包含：

1. 新异常事件音；
2. 目标或观察音型；
3. 主动作因果音；
4. 翻转前短静默；
5. 结果或翻转重音；
6. 可接下一集的未终止尾音。

模板：

```text
整体风格：无对白轻喜剧动作同步配乐。
主要配器：
0.0—0.8 秒异常：
0.8—2.2 秒目标：
2.2—5.5 秒主动作：
翻转前 0.15—0.25 秒静默：
结果 / 翻转：
下一集接点：
声音层级：
负面约束：
```

B 不要求每集都出现“第一次翻转 + 第二次翻转”两套重音。

## 3. C 模式完整声音

推荐曲线：

```text
强异常
→ 目标与蓄力
→ 第一动作加速
→ 完整静默
→ 第一次结果或翻转
→ 升级动作
→ 最终视觉/情绪回报
→ 未解决循环音型
```

### 3.1 异常事件

- 音乐立即进入，不播放平静前奏；
- 巨物体积和方向使用低音弦乐拨奏、低音管或圆润重音；
- 角色脚步和受惊使用木管、木块或短促上滑音。

### 3.2 目标与蓄力

- 弦乐拨奏逐步加速；
- 大力的认真和快速小步可用低音管短音；
- 啾啾锁定目标可用单簧管或长笛升高短音；
- 不提前出现高潮重音。

### 3.3 主动作

- 起跳、前冲、展开、回弹对应上行或加速音型；
- 落点使用轻拨弦、木块或柔和打击；
- 不使用沉重撞击。

### 3.4 静默

- 高潮前完整停止 0.15—0.25 秒；
- 只允许保留与因果直接相关的轻微声音；
- 静默期间无 BGM、对白和无关环境声。

### 3.5 结果和回报

- 结果第一帧有清楚短重音；
- 物体方向改变时，音乐方向或节奏同步改变；
- 最终回报使用更轻巧、清脆的音色；
- 最后 0.3 秒不使用完整终止和弦。
