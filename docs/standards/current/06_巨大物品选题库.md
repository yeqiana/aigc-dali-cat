## 2. 入选门槛

所有模式共同满足：

- 第一帧已经发生异常；
- 道具有明确可表演结构；
- 大力有可见目标或判断；
- 啾啾有自然行动或偏离；
- 无声能看懂主要因果；
- 高识别度主辅色；
- 无真实伤害、品牌和复杂文字；
- 题材指纹通过去重。

A-Lite 额外要求：

- 只有一个核心动作；
- 轮廓简单；
- 3—5 秒动态可表达；
- 4 次以内有较高稳定概率；
- 不依赖两次翻转。

B 额外要求：

- 对应动作家族已通过 A-Lite；
- 可形成 5 个不同节点；
- 同场景和道具可复用；
- 每个节点题材指纹不同。

C 额外要求：

- 已有免费数据；
- 有明确高潮或情绪回报；
- 付费工具能显著提升。

## 6. 评分

每项 0—2 分。

| 指标 | 0 | 1 | 2 |
|---|---|---|---|
| 第一帧 | 静态普通 | 巨大但静止 | 已发生明确异常 |
| 动作结构 | 无 | 单一但难读 | 简单、具体、可生成 |
| 角色关系 | 无 | 需要字幕 | 大力目标与啾啾偏离可见 |
| 结果 | 无 | 同动作继续 | 清楚结果、轻反转或循环 |
| 色彩 | 粘连 | 局部差异 | 强分离 |
| 生成稳定性 | 复杂 | 可降级 | 轮廓简单、4 次内可测 |
| 去重 | 高相似 | 有风险 | 指纹明显不同 |

A-Lite 至少 11/14 分；B/C 还必须通过对应升级门槛。
