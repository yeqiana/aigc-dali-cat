# 《啾啾和大力》通用场景母版验收标准 V1.1

> 生效日期：2026-07-11  
> 适用范围：通用室内场景控制包、单集固定场景母版，以及后续进入 B / C 模式的场景连续性参考。  
> 核心原则：场景母版只负责锁定空间，不代替角色母版、比例母版、表情母版或单集关键帧。  
> 验收原则：任何一项 P0 失败，当前图片或当前整组场景图结论必须为“不通过”。

## 2. 场景母版定位

通用场景母版用于锁定：

1. 房间身份；
2. 家具位置和结构；
3. 地面与空间透视；
4. 固定机位和镜头轴线；
5. 角色与巨物的可活动区域；
6. 主光方向、色彩和材质；
7. 不同关键帧和视频片段之间的空间连续性。

通用场景母版不得承担：

- 角色身份锁定；
- 角色表情锁定；
- 单集剧情说明；
- 多阶段动作演示；
- 已发布题材复刻；
- 用文字替代画面空间关系。

## 4. 状态定义

| 状态 | 含义 | 使用边界 |
|---|---|---|
| P0 已通过，强参考 | 所有对应 P0 均通过 | 可进入 `assets/masters/images/` 并作为正式生成强参考 |
| 仅构图或局部弱参考 | 只有部分空间信息可用 | 不得覆盖已通过场景强参考 |
| P0 未通过，候选 | 存在一项或多项 P0 失败 | 只能进入 `assets/masters/candidates/` |
| 历史场景，禁用于新内容 | 与旧剧情或旧空间强绑定 | 只作复盘证据，不作为新选题背景 |
| 按需生成 | 当前阶段不需要提前补齐 | 对应单集或系列被激活后再制作 |

## 5. 结论规则

场景母版验收只允许以下结论：

1. **通过**；
2. **不通过**。

不使用“基本通过”“勉强可用”“后期可以修”“问题不大”等模糊结论。

判定规则：

- 任意一个 P0 项目失败，当前图片结论必须为“不通过”；
- 多张图组成一个场景控制包时，任意一张出现空间身份冲突，整组结论必须为“不通过”；
- 画面好看、电影感强或细节丰富，不能覆盖空间漂移、轴线错误、家具换位和光线冲突；
- 仅可后期裁切的小范围边缘冗余，不得用于放宽 P0；
- 候选图文件名即使与正式母版相同，也不得覆盖已通过强参考。

## 6. 强参考优先级

同一场景信息发生冲突时：

```text
本对话中人工确认通过的当前场景图
> assets/masters/images/ 中 P0 已通过场景图
> 当前单集已通过关键帧
> S00 平面动线与机位图
> S02 空间尺寸标定图
> S03 光线色彩材质板
> 当前文字规范
> 候选场景图
> 旧分支和历史成片
```

补充规则：

- S00 负责平面位置和方向，不负责最终画风；
- S01 低机位空景负责最终可见空间和家具外观；
- S02 负责尺寸与高度；
- S03 负责光线、色彩和材质；
- 当不同控制图之间出现冲突时，必须先修正控制图，不得在单集提示词中临时解释。

# 第二部分：室内场景平面动线图专项补充

## 20. S00｜室内场景平面动线图的定位

S00 不是普通示意图，而是**所有通用室内场景图的上位控制图**。

它的核心任务是：

1. 锁定房间平面结构；
2. 锁定家具相对位置；
3. 锁定角色主要活动区；
4. 锁定巨物进入方向和活动方向；
5. 锁定固定机位；
6. 锁定镜头轴线；
7. 锁定安全区、遮挡区和禁止区；
8. 为 S01-A / B / C / D 空景、单集关键帧和视频动态提供统一空间依据。

S00 不负责：

- 直接作为正式成片背景；
- 表现电影感光影；
- 承担角色身份展示；
- 绘制单集剧情；
- 代替 S02 尺寸标定；
- 代替 S03 光线色彩材质板。

---

## 21. S00｜室内场景平面动线图必须表达的信息

一张合格的平面动线图，至少必须把下面信息表达清楚。

### 21.1 房间基础结构

- [ ] 房间边界；
- [ ] 墙面走向；
- [ ] 房门或玄关位置；
- [ ] 窗户位置；
- [ ] 地毯范围；
- [ ] 沙发位置；
- [ ] 低柜位置；
- [ ] 抽屉所在面；
- [ ] 落地灯位置；
- [ ] 绿植位置；
- [ ] 若有桌子、椅子，只能保留当前路线图需要的最小必要结构。

### 21.2 三大动作区

- [ ] A 区：地毯中心动作区；
- [ ] B 区：低家具 / 抽屉交互区；
- [ ] C 区：窗边 / 窗帘 / 光斑 / 气流交互区。

### 21.3 角色与巨物区域

- [ ] 角色默认待机区；
- [ ] 角色安全活动区；
- [ ] 巨物允许进入区；
- [ ] 巨物主要运动方向；
- [ ] 巨物可停留区；
- [ ] 家具遮挡区；
- [ ] 不可进入或不推荐进入区；
- [ ] 需要保留表情可读性的拍摄区。

### 21.4 机位和轴线

- [ ] C01：地毯正面低机位；
- [ ] C02：地毯左侧低机位；
- [ ] C03：低柜 / 抽屉机位；
- [ ] C04：窗边 / 窗帘机位；
- [ ] C05：玄关 / 滑入机位；
- [ ] 每个机位朝向箭头；

## 25. S00｜常见错误清单

以下问题非常常见，必须重点防止：

1. **只有平面，没有动线**  
   画了家具布局，但没有角色安全区、巨物进入方向和主要动作路线。

2. **只有动线，没有空间锚点**  
   画了很多箭头，但看不出沙发、低柜、窗户和地毯在哪里。

3. **动作区和家具重叠**  
   A 区画在沙发底下，B 区画进柜子里，C 区与窗帘位置对不上。

4. **机位乱放**  
   机位在墙内、家具后面或只能拍到背面。

5. **平面图和空景对不上**  
   平面图说窗户在左边，S01-C 却像窗户在右边。

6. **没有安全区**  
   只想着巨物怎么动，没有给角色留退路。

7. **越轴风险未标注**  
   结果同一题材不同关键帧方向反了，导致视频空间翻转。

8. **想一次解决所有系列**  
   画得过大、过杂、过复杂，最后谁也不能直接用。
