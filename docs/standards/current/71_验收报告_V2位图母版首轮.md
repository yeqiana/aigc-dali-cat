## 3. 使用边界

1. `images/` 内图片均可作为对应用途的强参考，但一张母版不得替代其他用途的专用母版。
2. C03、C04 只作为三视图辅助参考，不替代 C01、C02 的角色身份主母版。
3. M01 负责锁定双角色同框比例；其他双角色图片与 M01 冲突时，以 M01 为准。
4. M07 只参考动作关系和目标错位，不得复刻历史快递盒剧情。
5. 当前图片不用于强化真实宠物毛发、写实动物比例、静态低运动或暖棕低对比风格。
6. 单集仍需按题材补充专属巨物或关键帧，不能把通用母版直接当作成片。
