## 1. 母版体系目标

母版不只锁定“长什么样”，还要锁定：

- 动画化头身比例；
- 夸张表情边界；
- 预备、爆发、结果三类姿势；
- 巨物与背景的色彩分离；
- 不同景别下的角色可读性；
- 角色和道具连续性；
- 安全动作幅度。

## 5. 单张母版要求

每张母版必须：

- 只有一个明确用途；
- 保留角色身份、服装和比例；
- 标明建议画面比例，但成图内不出现文字；
- 使用至少三项具体材质或空间细节；
- 具有明确镜头距离、机位、光线和色彩方向；
- 不依赖外部作品名称或风格标签；
- 通过负面提示词限制文字、水印、品牌、额外角色和身份漂移。

## 6. 角色表情范围

### 啾啾六种核心表情

1. 突然受惊：瞳孔放大、耳朵竖起、身体后仰。
2. 锁定目标：身体压低、耳朵前倾、眼睛发亮。
3. 行动爆发：嘴角兴奋、四肢拉伸、轮廓前冲。
4. 无辜发愣：圆眼、嘴巴微张、身体僵住。
5. 满足入住：眯眼、身体摊平、前爪放松。
6. 小得意：轻微抬下巴、尾巴露出、眼神回看大力。

### 大力六种核心表情

1. 认真指挥：帽檐压低、双爪指向目标。
2. 拼命用力：眼睛挤紧、嘴巴咬紧、小脚打滑。
3. 突然受惊：双爪举高、嘴巴张开、身体后仰。
4. 发现翻车：瞳孔放大、帽子歪、全身僵住。
5. 无奈崩溃：肩膀下垂、嘴角下拉、双爪摊开。
6. 快速追赶：身体前倾、短腿高频移动、帽子保持唯一。

## 7. 动作安全边界

允许：

- 安全滑动、打滑、弹起、晃动、滚动；
- 短暂挤压和拉伸；
- 盒盖、帽子、绳子和把手参与喜剧动作；
- 角色被轻微遮挡后重新出现。

禁止：

- 真实撞伤、压伤、夹伤和摔伤；
- 角色恐惧、哭泣或疼痛作为笑点；
- 肢体扭曲、额外肢体和畸形爪子；
- 巨物突然变小或角色无因变大；
- 角色身份、服装和配饰漂移。
