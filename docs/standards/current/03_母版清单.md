## 4. 强参考优先级

同一内容发生冲突时：

```text
本对话中人工确认通过的当前图片
> assets/masters/images/ 中 P0 已通过图片
> 当前单集已通过关键帧
> 当前规范文字
> 候选母版
> 旧分支和历史成片
```

候选图不得因为文件名相同而覆盖强参考。

---

## 5. 单张母版 P0

- [ ] 只有一个明确用途；
- [ ] 啾啾身份、毛色、脸型和背心正确；
- [ ] 大力身份、比例和唯一蓝帽正确；
- [ ] 两者处于同一透视地面；
- [ ] 巨物 > 啾啾 > 大力；
- [ ] 角色轮廓无畸形和额外肢体；
- [ ] 表情在手机尺寸可读；
- [ ] 巨物与背景有色彩或明暗分离；
- [ ] 无文字、水印、品牌、完整人类和额外动物；
- [ ] 无真实伤害和危险感；
- [ ] 不命中禁重复清单。

任一项失败，不进入 `assets/masters/images/`。

---

## 6. 母版生成原则

- 不为“资产完整”而批量生成；
- 优先服务当前正在制作的单集；
- 一个母版只解决一个问题；
- 历史禁重复题材的母版保留，不继续扩写；
- M11 毛线球可以用于材质和比例，但不得出现帽子被线带走；
- 新道具先做单集首帧，数据胜出后再升级为通用母版；
- 免费额度阶段优先把额度投入正式可发布内容。

---

## 7. 文件管理

```text
assets/masters/images/        仅 P0 通过强参考
assets/masters/candidates/    未通过或待校准图片
episodes/<slug>/images/       单集关键帧
```

只有通过 P0 的图片才能从 `candidates/` 移入 `images/`。
