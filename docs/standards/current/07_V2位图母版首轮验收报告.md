## 3. P1 观察

- 整体风格能识别为 3D 动画角色，不是现实宠物照片。
- 角色服装、毛色、帽子和环境暖色基调较稳定。
- M05、M07、M15 的动作方向可读，适合作为返修构图参考。
- M10 已能识别为拖鞋，但轮廓仍略接近宠物窝，返修时建议强化鞋底、鞋口和前后方向。

## 4. 返修要求

1. 只返修 M01–M12、M15、M16，不重生成已通过项。
2. 每条提示词增加硬约束：大力全身高度约为啾啾肩高的 40%，禁止因透视或前景位置放大。
3. 双角色尽量落在相近景深平面，避免用透视制造假比例。
4. 返修时引用 C01、C02、C03、C04 锁定身份，再引用原失败图锁定构图；不得把失败图作为角色比例参考。
5. 新图仍按相同 P0 清单复验，通过后才能移回 `assets/masters/images/`。
