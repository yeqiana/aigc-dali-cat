# 《啾啾和大力》强节奏 V3 → 分级生产 V4 迁移说明

> 生效日期：2026-07-11  
> 目标：解决“所有单集都被强制高成本制作”的冲突。

---

## 1. 核心变化

| V3 强节奏体系 | V4 分级体系 |
|---|---|
| 默认 8—10 秒 | A 6—8；B 8—12；C 10—15+ |
| 默认 4—6 个短片段 | A 1；B 1—2；C 2—4 |
| 默认两次翻转 | A 轻反转/循环；B 1—2；C 完整回报 |
| 全部完整声音正文 | A 简化；B 标准；C 完整 |
| 共用一份单集模板 | 02A / 02B / 02C |
| 所有题材同一验收 | 09 分模式验收 |
| 先补完整母版 | 按当前单集需要补 |
| 路线图 V2.1 | 去重路线图 V2.2 |
