## 1. 题材指纹

统一使用：

```text
道具 + 核心运动 + 大力目标 + 啾啾偏离 + 最终结果
```

辅助维度：

```text
场景 + 运动方向 + 第一帧轮廓 + 角色站位 + 结尾姿势
```

---

## 2. 相似判定

### 直接不通过

- 命中永久禁区；
- 完整题材指纹相同；
- 只是更换道具颜色、背景或标题；
- 只是把同一动作方向左右镜像；
- 只是延长时长或提高画质；
- 蓝帽再次成为任务道具。

### 高相似不通过

核心五项中有三项及以上相同：

1. 道具；
2. 核心运动；
3. 大力目标；
4. 啾啾偏离；
5. 最终结果。

### 可复用条件

同一道具允许在 B/C 复用，但必须：

- 相关 A-Lite 数据胜出；
- 不重新演一遍 A-Lite 首次异常；
- 核心运动、冲突和结果至少改变两项；
- 明确属于五集连续故事的新节点；
- 再次通过本清单。

A-Lite 前 30 集原则上不重复道具；确需重复时，至少间隔 10 条，且核心运动、冲突、结果全部不同。

## 3. 已确认发布/已使用内容

### PUB-001｜帽子搭便车

| 字段 | 内容 |
|---|---|
| 道具 | 巨大毛线球、毛线、蓝帽 |
| 核心运动 | 毛线拉扯、帽子被带动 |
| 大力目标 | 拉住或保护帽子 |
| 啾啾偏离 | 追逐线头或继续推动动作 |
| 最终结果 | 帽子被带走、转移或产生追帽 |
| 状态 | 永久禁重复 |

禁止变体：

- 帽子被丝带、绳子、胶带、气流、雨伞、纸带或任何巨物带走；
- 大力追帽子；
- 帽子落到啾啾头上作为结尾；
- 帽子成为车、船、滑板、降落伞、篮子；
- 用不同颜色绳线重做相同剧情；
- 毛线球再次承担“帽子被拖走”功能。

M11 毛线球母版只允许参考材质和比例。

### PUB-002｜巨大西瓜

| 字段 | 内容 |
|---|---|
| 道具 | 巨大西瓜 |
| 核心运动 | 圆形巨物滚动 |
| 大力目标 | 阻止或改变滚动 |
| 啾啾偏离 | 把巨物当球追逐 |
| 最终结果 | 追逐、打转或巨物停下后的继续运动 |
| 状态 | 永久禁重复 |

禁止变体：

- 哈密瓜、南瓜、橙子、柚子等只换水果；
- 巨大圆球滚入，啾啾追，大力阻止；
- 地毯隆起后圆形水果滚动；
- 结尾啾啾继续追或打转。

允许其他圆形物品的前提：

- 不是水果；
- 不使用追逐冲突；
- 运动结构和结果完全不同；
- 通过人工去重。

### PUB-003｜快递盒为家

| 字段 | 内容 |
|---|---|
| 道具 | 快递盒/纸箱 |
| 核心运动 | 箱体进入、开合或形成空间 |
| 大力目标 | 搬走、推出或处理纸箱 |
| 啾啾偏离 | 把纸箱当家、卧室或入住 |
| 最终结果 | 啾啾进入、躺下、探头或长期占据 |
| 状态 | 永久禁重复 |

禁止变体：

- 礼物盒、收纳箱、购物袋仅换外观后入住；
- 啾啾钻入容器睡觉；
- 啾啾从箱口探头表示“这是我的家”；
- 大力想清理，啾啾直接入住；
- 用快递盒重制旧剧情。

允许容器类道具的前提：

- 不进入、不入住、不睡觉；
- 核心动作是外部开合、旋转、滑动或形变；
- 角色始终在外部；
- 结果不是从孔洞探头；
- 通过人工去重。

## 4. 永久内容边界

### 蓝帽

蓝帽只用于：

- 身份识别；
- 表情角度；
- 轻微歪斜；
- 连续性检查。

蓝帽不得：

- 被带走；
- 丢失；
- 被寻找；
- 被争抢；
- 被保护为整集目标；
- 成为交通工具；
- 成为容器；
- 成为机关；
- 成为悬念线索。

### 配角

前 30 集不得引入小兔、乌鸦或其他配角。

### 伤害

不得通过真实碰撞、夹伤、压伤、困住、疼痛、哭泣或恐惧制造笑点。

## 7. 发布后更新流程

每条发布后：

1. 将对应 `PLAN` 状态改为 `已发布`；
2. 记录发布日期和实际题材指纹；
3. 若成片和计划发生变化，以实际成片为准；
4. 补充第一帧和结尾姿势；
5. 下一条立项时重新计算最近 10 条；
6. 发现未登记旧内容时，先补登记再立项。
