# 《啾啾和大力》AI 短视频资产库

本仓库管理《啾啾和大力》的当前生产规范、视觉母版、战略路线图、单集执行表、复盘和交付资产。

---

## 当前体系

当前使用 V4.0 分级生产体系：

- A-Lite：6—8 秒，1 段动态，1—2 张关键帧，验证新题材；
- B：8—12 秒，1—2 段动态，五集连续故事；
- C：10—15 秒以上，2—4 段动态，只重制数据胜出题材。

生产新内容从：

```text
docs/standards/current/00_当前版本与执行入口.md
```

开始。
