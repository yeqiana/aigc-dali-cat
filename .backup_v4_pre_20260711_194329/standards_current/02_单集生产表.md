## 12. 声音设计

> 本节执行 `07_配乐与音效执行规范.md`。下表填写完成后，必须把完整内容原样合并进最终视频提示词。

| 时间 | 画面动作 | 管弦乐配器与节奏 | 因果音效 | 静默/重音/循环 |
|---:|---|---|---|---|
| 开头事件 |  |  |  | 强事件重音，音乐立即进入 |
| 目标行动 |  |  |  | 轻快蓄力并逐渐加速 |
| 主动作 |  |  |  | 第一次速度峰值 |
| 翻转前 |  |  |  | 管弦乐截断并完整静默0.15–0.25秒 |
| 第一次翻转 |  |  |  | 错位重音，音乐方向或节奏改变 |
| 第二次翻转 |  |  |  | 三次递进短音，笑点快速收尾 |
| 循环接点 |  |  |  | 未解决音型接回开头，不用完整终止和弦 |

### 12.1 动作同步原则

```text
巨物速度或方向变化：
角色预备动作：
角色爆发动作：
动作落点：
高潮前截断：
第一次翻转重音：
第二次翻转递进：
结尾反应音：
```

### 12.2 声音层级

```text
关键因果音效 > 高潮前完整静默 > 动作同步管弦乐 > 极弱环境底噪
```

### 12.3 配乐负面约束

```text
不要流行歌曲、歌词、人声演唱、旁白、对白、电子舞曲、史诗预告片音乐、恐怖弦乐、持续轰鸣低音、全程同速、用音乐填满静默、沉重伤害音效、疼痛叫声和破坏循环的完整终止和弦。
```

必须包含动作同步管弦乐、明确因果音效、0.15–0.25秒完整静默、翻转重音和未解决循环尾音；最终峰值不高于 -1 dBFS。

## 14. P0 发布验收

- [ ] 第一帧已经发生异常事件。
- [ ] 0.6 秒内能识别巨物和角色反应。
- [ ] 前 3 秒无声观看能理解目标和冲突。
- [ ] 每个片段只承担一个主动作或主揭示。
- [ ] 每个动作片段都有预备、爆发和结果状态。
- [ ] 相邻镜头改变了故事状态，而不只是景别。
- [ ] 啾啾和大力均有明显身体表演。
- [ ] 角色不像真实宠物摄影。
- [ ] 巨物与背景具有色彩或明暗分离。
- [ ] 比例、服装和唯一帽子保持连续。
- [ ] 结尾包含第二次翻转或循环接点。
- [ ] 最终视频提示词包含完整配乐与音效时间轴，不是简短声音备注。
- [ ] 动作同步管弦乐、关键因果音效、高潮前完整静默、翻转重音和循环尾音全部明确。
- [ ] 全片无对白、旁白、歌词和无意义叫声，配乐不遮挡关键因果音效。
- [ ] 成片没有无信息停顿。
- [ ] 字幕服务冲突，不重复无意义叫声。
- [ ] 导出规格符合要求。
