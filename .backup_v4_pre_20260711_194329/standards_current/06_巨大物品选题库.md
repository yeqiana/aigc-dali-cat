# 《啾啾和大力》巨大物品选题库 V2.0｜强节奏版

## 1. 入选标准

一个选题必须同时满足：

- 第一帧能形成巨大轮廓或运动事件；
- 物品至少有一个可表演结构；
- 大力能建立明确任务；
- 啾啾能自然偏离目标；
- 能设计第二次翻转；
- 有区别于背景的高识别度颜色；
- 不依赖危险、伤害或复杂文字。

## 2. 评分

每项 0–2 分，总分至少 9 分才进入生产：

| 指标 | 0分 | 1分 | 2分 |
|---|---|---|---|
| 第一帧 | 静态普通 | 巨大但静止 | 巨大且正在发生事件 |
| 动作结构 | 无 | 单一移动 | 可弹、滚、折、拉或遮挡 |
| 目标错位 | 不清楚 | 需要字幕 | 无声可读 |
| 第二翻转 | 没有 | 较弱 | 清晰且可循环 |
| 色彩 | 与背景粘连 | 有局部差异 | 强主辅色分离 |
| 生成稳定性 | 结构复杂 | 可控制 | 轮廓简单、动作明确 |

## 5. 暂缓选题

以下物品在当前阶段不优先：

- 刀具、玻璃、电器和明火；
- 液体大量倾倒；
- 结构过于复杂的机械物品；
- 需要品牌或文字才能识别的物品；
- 只能静态展示、缺少动作结构的物品；
- 容易造成角色受伤联想的重物。

## 6. 生产顺序

1. 先验证快递盒强节奏重制。
2. 再使用雨伞或毛线球验证不同运动结构。
3. 第三条使用纸袋或礼物盒验证遮挡与揭示。
4. 连续三条数据稳定后，再扩展 P1 选题。

每个新选题先填写 `02_单集生产表.md` 的前 3 秒钩子和九格故事板，通过评审后才生成关键帧。
