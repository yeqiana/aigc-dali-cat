# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

《啾啾和大力》AI 短视频系列资产库。主角是白色小猫（啾啾）和戴蓝色牛仔帽的小仓鼠（大力），定位为巨大日常物品 + 居家轻冒险 + 无伤害轻喜剧。

## 目录结构

```
assets/masters/images/        P0 验收通过的视觉母版（C01-C04 + 校准图）
assets/masters/candidates/    未通过验收的候选母版（v2-p0-failed/、v2-contact-sheet-crops/）
docs/
  standards/current/          生产规范（编号 00-99，00 为执行入口）
  strategy/                   IP 路线图与战略层文档
episodes/<episode-slug>/      单集目录
  docs/                       单集执行表、复盘
  images/                     关键帧
  video/                      交付视频
```

## 文档编号体系（docs/standards/current/）

| 编号 | 用途 |
|------|------|
| `00_` | 执行入口（唯一起点） |
| `01_`–`09_` | 活动规范（仅 01-09 为生产必用） |
| `71_`–`79_` | 验收报告 |
| `99_` | 版本迁移说明 |

- 规范编号不降级：旧规范归档到 Git 分支 `1.0`，不在 main 分支维护多版本。
- 新单集从 `02_单集生产表.md` 复制模板，不从旧分支复制。

## 角色核心约束（P0，不可修改）

- 啾啾：白色小猫，始终穿粉色背心。
- 大力：戴牛仔蓝帽子（唯一配饰，不作为道具/交通工具），约为啾啾肩高 40%。
- 关系：搭档，非捕食关系。大力负责计划，啾啾负责行动，笑点来自目标错位。
- 安全边界：无伤害、无完整人类、无品牌、无对白旁白歌词、无血腥恐怖。

## 生产方式

- 最终执行规范：`01_最终执行规范.md`（强节奏动画锁定版 V3.0）
- A-Lite 模式：6-8 秒、1 段动态、1-2 张关键帧、1 个核心动作（见 `08_`）
- 视频配方：第一帧已异常 → 前 3 秒可见目标和冲突 → 两次翻转/循环结尾
- 配乐：复古无对白动画喜剧的管弦乐，包含动作同步点、高潮前静默、循环尾音（见 `07_`）
- 每条视频提示词必须包含完整配乐与音效时间轴，禁止"轻快音乐"式简写。

## 母版管理

- `images/` 只保留 P0 验收通过的母版。
- 未通过者移入 `candidates/` 对应子目录，不删除。
- 新母版按 `04_全量母版生成提示词.md` 生成，通过 P0 验收后才作为视频强参考。
- 编号约定：`C01-C04`（角色设定），`M01-M16`（场景/动作母版）。

## 单集目录约定

新建单集按 `episodes/<episode-slug>/` 组织：
- `docs/` — 执行表、复盘文档
- `images/` — 关键帧图片
- `video/` — 交付视频

## Git 约定

- 当前分支 `main`（强节奏动画生产体系），旧版快照在分支 `1.0`。
- 提交信息格式：
  - 新增规范/母版：`新增<标题>`
  - 更新现有内容：`升级/整理<模块>：<变更说明>`
- `.codex/` 已加入 `.gitignore`，不提交工具输出。
