# 视觉母版资产状态

当前 `images/` 只保存通过 V2 位图 P0 验收、可作为视频强参考的图片。

截至 2026-07-11 的当前状态：

- 正式母版通过 9 张：C01、C02、M01、M02、M04、M05、M07、M13、M14；
- 辅助三视图通过 2 张：C03、C04；
- 首轮未通过图保留在 `candidates/v2-p0-failed/`，只作为待返修候选；
- 接触表裁切源和生成过程资料归档到 `history/v2-contact-sheet-crops/`，不作为当前强参考。

通过项用于：

- 角色身份和服装参考；
- 啾啾与大力体型关系参考；
- 玄关、地毯和桌边空间构图参考；
- 巨大物品基础比例参考。

这些图片不再作为以下内容的强参考：

- 真实宠物毛发；
- 写实动物头身比例；
- 静态表情；
- 全程低运动；
- 暖棕低对比色彩。

目录边界：

- `images/`：当前 P0 已通过强参考；
- `candidates/`：未通过或待校准图片；
- `history/`：生成源、过程稿和历史证据。

新图片通过母版 P0 验收后，按 C01、C02、M01–M16 编号替换或补充当前资产。未通过验收的生成图不得作为视频强参考。
