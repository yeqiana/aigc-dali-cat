## 3. 立项与禁重复

### 3.1 题材指纹

```text
道具：巨大牛皮纸袋
核心运动：鼓起、缓慢滑动、泄气停止
大力目标：判断气流方向并让纸袋安全停下
啾啾偏离：不等大力判断完成，直接上前按住袋面
最终结果：纸袋泄气瘪下并停止，提手仍轻轻颤动

完整指纹：
巨大牛皮纸袋
+ 鼓起与缓慢滑动
+ 大力判断气流并让纸袋停下
+ 啾啾直接按住袋面
+ 纸袋泄气停止且提手轻颤
```

### 3.3 差异说明

- 纸袋不是帽子、交通工具或任务道具；
- 纸袋不是水果类圆形巨物，不使用追逐；
- 啾啾不进入纸袋，不把纸袋当家或卧室；
- 笑点来自“大力仍在判断，啾啾已经直接行动”；
- 结尾是泄气和提手轻颤，不是继续追逐或再次事故。

```text
奶茶色小男仓鼠；
```

```text
大力负责判断和指挥；
啾啾负责直接行动；
```

- 纸袋必须是柔软可鼓起、可泄气的牛皮纸袋；

- 纸袋泄气不能表现成爆炸；

## 9.2 KF02｜结果帧（建议制作）

```text
用途：
明确“纸袋泄气并停止”的结果，为视频尾帧和静态降级方案提供强参考。

结果状态：
纸袋已经泄气瘪下；
袋体停止滑动；
两个提手仍轻轻颤动；
没有爆炸和飞散纸屑。

与 KF01 的连续关系：
同一场景；
同一镜头方向；
同一纸袋颜色、材质和提手结构；
纸袋由鼓起滑动变为瘪下停止；
角色站位只发生必要的小幅变化。

角色比例：
保持啾啾 > 大力；
大力约为啾啾肩高 35%—45%；
不能通过远近透视伪造比例。

巨物变化：
袋面张力消失；
体积明显缩小；
纸张折痕和塌陷褶皱增加；
提手有轻微余震。

表情变化：
啾啾保持前爪按住袋面，专注或轻微满意；
大力从认真判断变为轻微无奈，但不愤怒、不恐惧。

循环接点：
提手轻颤；
可用极短纸张抖动音作为尾音；
不要引入第二次事故。

负面约束：
不要爆炸；
不要袋体飞起；
不要纸袋罩住角色；
不要增加追逐；
不要帽子被带走；
不要新角色和新道具。
```

### 是否需要 KF02

```text
建议：是

2. 可提高视频首尾连续性；
3. 视频生成失败时，可降级为 KF01 + KF02 静态剪辑；
4. 提手轻颤可作为循环尾帧。
```

## 14. 四次尝试止损

| 尝试 | 只允许处理的事项 | 通过标准 |
|---:|---|---|
| 1 | 先保证纸袋鼓起、滑动方向、啾啾按住三个核心状态 | 唯一动作因果成立 |
| 2 | 如角色漂移，减少大力运动，只让大力观察 | 身份和比例稳定 |
| 3 | 如动态混乱，降低纸袋速度、缩短位移、固定啾啾动作 | 纸袋泄气结果清楚 |
| 4 | 最后一次，仅修最严重的一项，不增加新动作 | 仍失败则立即停止 |

### 第 4 次失败后的降级方案

- [ ] 使用 KF01 + KF02 两张静态帧；
- [ ] 用 2D 推镜或轻微缩放表现纸袋滑动；
- [ ] 用局部形变或遮罩表现纸袋泄气；
- [ ] 用纸袋摩擦声和泄气声补足中间过程；
- [ ] 保留提手轻颤作为尾帧；
- [ ] 不继续消耗免费或付费额度；
- [ ] 记录失败原因后进入下一候选题材。
