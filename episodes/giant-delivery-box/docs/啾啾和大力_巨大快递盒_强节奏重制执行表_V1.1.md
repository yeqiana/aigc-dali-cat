## 2. 视觉摘要

### 2.1 情绪核心

大力越认真，啾啾越按照自己的理解行动。喜剧不是来自危险，而是来自目标错位：一个想搬走，一个直接入住。

### 2.2 关系状态

- 啾啾和大力始终是搭档，不是猫抓鼠关系。
- 大力负责计划和指挥，啾啾负责快速行动和温和翻车。
- 大力的崩溃只表现为夸张无奈，不表现恐惧或受伤。

### 2.3 感官锚点

- 粗糙瓦楞纸板摩擦声；
- 青绿色胶带在暖棕纸箱上的强色彩切割；
- 盒盖弹起的清脆纸板声；
- 大力快速打滑的小脚步声；
- 纸箱晃动后突然安静；
- 帽子被拖入缝隙的短促摩擦声。

### 2.4 关键道具

- 巨大无品牌快递纸箱；
- 一条宽青绿色封箱胶带；
- 大力唯一一顶低饱和牛仔蓝棒球帽；
- 浅米色编织地毯；
- 玄关浅木门和简洁鞋柜。

### 2.5 画面风险

- 纸箱、地板和木门再次混成同一棕色；
- 啾啾和大力变成写实宠物；
- 大力帽子复制或脱离角色连续性；
- 纸箱运动变成危险撞击；
- 一个片段出现多个无关动作；
- AI 自动生成文字、快递单或品牌标识。

## 3. 角色卡

### 3.1 啾啾

- 身份：白色银渐层小母猫。
- 比例：四足肩高约 24 cm，头部偏大，四肢短圆。
- 服装：粉色绒绒小背心，白色包边。
- 动画特征：大而清晰的深色眼睛，圆脸，柔软块状绒毛，耳朵和身体重心参与表演。
- 常见姿势：好奇压低身体、跳跃时身体拉长、满足时摊平眯眼。
- 性格：好奇、先行动、闯祸后无辜。
- 不可变化：不换衣服、不换品种、不增加项圈和多余配饰、不使用真实成年猫头身比例。

### 3.2 大力

- 身份：奶茶色小男仓鼠。
- 比例：约为啾啾肩高的 40%，含帽最高不超过啾啾肩高的 50%。
- 服装：只戴一顶低饱和牛仔蓝小棒球帽。
- 动画特征：圆头、短手短脚、身体小而有弹性，眉眼和嘴型清晰。
- 常见姿势：双爪指挥、顶住物体用力、小脚快速打滑、崩溃时全身僵住。
- 性格：认真、爱指挥、努力解决问题。
- 不可变化：不变大、不变成老鼠、不长尾巴、不穿其他衣服、不出现第二顶帽子。

### 3.3 比例锁定

```text
巨大快递盒 > 啾啾 > 大力。
大力约为啾啾肩高的 40%。
纸箱开口足以让啾啾轻松进入，但不制造被卡住的危险感。
```

## 9. 剪辑执行表

| 接点 | 处理 |
|---|---|
| C01 → C02 | 在两只后仰到最大幅度时切到大力顶箱首帧 |
| C02 → C03 | 在啾啾压低身体的尾帧直接动作匹配到起跳首帧 |
| C03 → C04 | 在啾啾完全消失、大力扑到箱口时切到纸箱晃动 |
| C04 → C05 | 啾啾满足眯眼后保留约 0.25 秒，再切到抬爪关门 |
| C05 → C06 | 盒盖只剩窄缝时切到大力扶帽，保持帽子位置连续 |
| C06 → 循环 | 大力向左扑出的运动模糊接回纸箱从左向右进入，可加入 2–3 帧快速遮挡 |

剪辑要求：

1. 不保留任何只展示场景的空镜。
2. 总时长优先控制在 8.5–9.5 秒。
3. 前 3 秒必须包含纸箱进入、大力目标和啾啾偏离。
4. C04 的静默是全片主要节奏停顿，不再额外拉长其他反应。
5. `meow` 等无叙事字幕不进入成片。
6. 如需文字钩子，仅在剪辑阶段加入一句：`让她搬箱子，她直接入住了`。

## 12. P0 验收

- ☐ 第一帧纸箱正在运动，两只角色已经产生身体反应。
- ☐ 0.6 秒内能识别纸箱、啾啾和大力。

- ☐ 纸箱从封闭到打开具有盒盖弹起的明确因果。
- ☐ 六个片段分别只承担一个主动作或主揭示。
- ☐ 啾啾至少有后仰、压低、跳跃、满足四种明显姿势。
- ☐ 大力至少有惊吓、推箱、追赶、僵住、扑帽五种明显姿势。
- ☐ 大力始终只有一顶帽子，角色比例稳定。
- ☐ 角色具有动画电影感，不像真实宠物摄影。
- ☐ 青绿色胶带能把纸箱与暖棕背景分开。

- ☐ 无无效 `meow` 字幕、品牌、快递单和水印。
- ☐ 成片总时长在 8.5–9.5 秒。
- ☐ 导出达到 1080 × 1920、H.264、8 Mbps 以上。
